Study of Sentences

An exploration of complete sentences as operators acting on invariant structure. This project collects definitions, operators, and worked sentence analyses.

Overview Sections Manuscript How to read this GitHub Repo

Overview

The core idea of this project is that a fully formed sentence does not just state a fact: it encodes an operator–invariant structure. The materials here are arranged so you can move between the abstract machinery (invariants and operators) and concrete sentence-level studies.

The repository is organized into a small number of parts: invariants for the stable structures, operators for the transformations or relations between them, and sentence studies where the framework is exercised on real examples.

How to read this project

Depending on your background, you might prefer to start from abstractions, from examples, or from the manuscript.

  • If you like formal structure first, start with Invariants → Operators → Sentence studies.
  • If you like concrete examples first, begin with Sentence studies and only then circle back to the invariants and operators.
  • If you prefer a single narrative, read the manuscript straight through, and treat the folders as reference material.
